📋 Anleitung GitHub Gist

1. Gehe zu github.com/settings/tokens
2. „Generate new token (classic)"
3. Haken bei gist setzen
4. Token kopieren und hier einfügen
5. Iris macht das Gleiche mit dem gleichen Token

📋 Anleitung JSONBin.io

1. Gehe zu jsonbin.io → kostenlos registrieren
2. Dashboard → API Keys → Master Key kopieren
3. Iris: Selben Key + Bin-ID verwenden
